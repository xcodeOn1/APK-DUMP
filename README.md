# APK-DUMP
APK-DUMP Script dump android app be ssh .

# what new in this script :
- without USB .
- you don't need use ADB shell .
- very fast.

# how use it ?
install openssh in termux :
- pkg install openssh .

# Run APK_DUMP
python3 APP-DUMP.py -t you ip -p were you save apk dump
